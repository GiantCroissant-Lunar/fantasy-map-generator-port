## MorphologyRules Features

- **Morpheme Database** - Tracks semantic → phonetic mappings
- **Affix System** - Prefix, suffix, and infix support with frequency control
- **Compounding** - Combine words with configurable joiners and head-first/final
- **Morphophonemic Rules** - Sound changes at morpheme boundaries
- **Deterministic Generation** - Same seed produces same morphemes
- **Clone Support** - Create variants with new seeds

## Components

### Affix
- Form (phonetic shape)
- Meaning (semantic value)
- Frequency (0.0-1.0)
- Position (Prefix/Suffix/Infix)

### CompoundingRules
- Joiner (space, hyphen, empty)
- HeadFirst (head-initial vs head-final)
- CompoundProbability (0.0-1.0)
- MaxCompoundLength

### MorphophonemicRule
- Pattern (regex)
- Replacement
- Context-aware application
