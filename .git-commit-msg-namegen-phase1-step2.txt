feat(namegen): Implement Layer 2 - CulturePhonology system

Completed Phase 1, Step 2 of the name generation system.

## What's New

- Implemented `CulturePhonology` class (Layer 2 - Phonology)
- Created `PhonemeInventory` for organizing sound systems
- Added `AllophoneRule` for contextual sound changes
- Built 6 predefined phonology templates
- Added comprehensive unit tests (28 new tests, 47 total passing)

## CulturePhonology Features

- Phoneme inventory management (consonants, vowels, liquids, nasals, fricatives, stops, sibilants, finals)
- Phoneme frequency weights for realistic distribution
- Allophonic rules for contextual sound changes
- Orthography system (IPA → written form)
- Phoneme shuffling for language mutation
- Clone support for creating language variants

## Phonology Templates

Built-in templates based on real-world and fantasy languages:

1. **Germanic** - Hard consonants, consonant clusters (English, German, Norse)
2. **Romance** - Flowing vowels, softer consonants (Latin, Italian, Spanish)
3. **Slavic** - Consonant clusters, palatalization (Russian, Polish, Czech)
4. **Elvish** - Melodic, flowing, lots of liquids (Tolkien-inspired)
5. **Dwarvish** - Harsh, guttural, consonant clusters
6. **Orcish** - Simple, brutal, minimal vowels
